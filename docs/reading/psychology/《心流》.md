> 当被情绪控制时，该如何摆脱？

<!-- more -->

《心流》提出的“心流状态”理论，至今已被广泛应用于教育、管理、运动等诸多领域，对人类幸福感的研究具有里程碑意义。

《心流》最初出版于1990年，由心理学家米哈里·切克森米哈赖提出，是正向心理学奠基性著作之一。心流理论被视为“最具影响力的心理学发现”之一，并已在教育、运动和管理领域广泛应用。运动员、艺术家和高管都在不同场景中报告了心流状态对绩效与幸福感的积极影响。

它告诉我们：**感受幸福，是一门人人都能学会的技能。**

## 什么是“心流”？

“心流”是心理学之父米哈里·切克森米哈赖在三十年前提出的概念，它指我们在做某些事时那种全神贯注投入忘我的状态，在这种状态下我们感觉不到时间的存在，而且做完这件事之后我们会有一种充满能量且非常享受的感觉。

很多人认为，生命中最美好的时光某过于心无牵挂、完全放松的时刻。可米哈里不这么认为，他认为人最愉快的时刻是在他为了某项任务辛苦付出，把体能和智力都发挥到极致的时刻。

比如对一个婴儿来说，也许是把最后一块积木放到他从未堆到的高度的时候；对一个游泳运动员来说，也许是刷新他自己创下的记录的时候；对一个音乐家来说，也许是一复杂的乐曲演奏得出神入化的时候。

许多人误以为放松时最快乐，切克森米哈赖指出，**真正的幸福感来自对挑战的全情投入**。

米哈里认为<HSentence left>每个人一生都面临着不计其数的挑战，而每一次挑战。都是一个获得幸福的良机。</HSentence>

_那么，如何把挑战变成幸福？心流又是怎么形成的呢？_

## 心流的产生

关于心流，我们需要了解5个要素:

### 要素一：资讯

我们感知到的所有信息，以及我们所有的经历都是资讯，我们每天都会接触数以百万计的资讯。

比如早晨看到的风景、耳边的噪音、昨晚看到的电影等等。

### 要素二：意识

意识的功能是搜集和组织资讯，加以评估，以便让身体做出反应。

比如我们一天没吃饭，身体出现血糖很低这个资讯，于是我们产生了饥饿这个意识。

### 要素三：意图

意图来自我们的生理需求或目标，它能帮助我们的意识去筛选和整理资讯。

比如，我们产生想让自己身体更好的意图，这个意图让我们的意识更关注和健身相关的资讯。

### 要素四：注意力

注意力就是专注力。我们的意识屏蔽干扰资讯的能力越强，专注力就越高。

比如我们在工作时，外界嘈杂的环境有碍于我们达到工作目标，我们的意识自动把它屏蔽了，我们就更能专注地达成目标。

### 要素五：及时反馈

在我们做事的过程中，当意识和意图高度一致，大量的无效资讯就会被屏蔽，我们的注意力就能持续不断的接收到靠近目标的及时反馈。这个及时反馈，会让我们欲罢不能的沉浸其中，心流就形成了。

---

可是，人类的意识处理资讯的能力相当有限，当外界资讯过多时，意识和意图会发生冲突，给我们的内心造成失序。失序的表现就是痛苦、恐惧、愤怒、焦虑、嫉妒等等。

米哈里称，这种内心失序的现象为“精神熵”^[“熵”指“熵增”。科学里解释：熵增的过程是一个自发的由有序向无序发展的过程。]。

## 进入心流

数据显示当我们开始做一件事时，花15分钟时间去控制自己的意识，让收到的资讯与目标亲和，精神能量就会源源不断的进入，帮助我们进入心流。这个状态下，我们就能够自动去除那些干扰我们的资讯，就能持续处于心流状态。而这，正好是“精神熵”的反面，米哈里叫它“精神负熵”。

| 特征   | 精神熵   | 精神负熵  |
| ---- | ----- | ----- |
| 资讯量  | 过多、紊乱 | 受控、聚焦 |
| 情绪体验 | 焦虑、挫败 | 活力、满足 |
| 状态   | 分心、疲惫 | 心流、愉悦 |


所以，进入心流只需要三个步骤：
- 第一步：**设定挑战性目标**，创造清晰明确的目标，这个目标最好有一点挑战性。
- 第二步：**屏蔽无关干扰**，花一些时间屏蔽干扰资讯，控制意识和意图保持一致。
- 第三步：**捕捉及时反馈**，不断去接收及时反馈，进入心流。

米哈里认为<HSentence>一个人控制意识的能力决定了他的生活品质。</HSentence>

在工作中利用心流，能让我们幸福高效的产出结果；在亲密关系中，如果两个人能够一起明确目标，共同排除干扰资讯，及时发送和接收即时反馈，就能让一段关系长久保持亲密。
