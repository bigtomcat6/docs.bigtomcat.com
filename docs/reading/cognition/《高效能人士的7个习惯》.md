这本书畅销了30多年，畅销4000多万册的书，被《福布斯》杂志评价为“有史以来最具有影响力的十大管理书籍”之一，科维的理念至今仍被全球顶尖企业与个人广泛践行。

作者史蒂芬·科维，他的“7个习惯”思想是一套让很多人立于不败之地的制胜法则。

## 习惯一：积极主动

> 人为什么能成为万物之灵，为什么一部分人能凌驾于另一部分人？

简单来说，是我们对抗“原始人性”的意识强度。举个例子：一个人被打了一顿，反击之前，人会下意识地思考“他为什么打我？”，一只猎犬被打了一棍，它会立即冲过去咬对方一口。

所以自我意识很强的人，能够更加自由地选择回应外界刺激的方式。

_那么，它与积极主动有什么关系？_

每个人都会面临许多需要关注的问题，而只有少数自我意识强的人才能真正区分“想做的事”和“有能力做的事”。如果把所有 我们关注的事，都归入一个“关注圈”，把我们能力范围内的事，归入一个较小的“影响圈”。

- 自我意识强的人总能排除外界干扰，专注于自己能力范围内的事，这种积极的能量能逐渐扩大他们的影响圈。
- 自我意识弱的人，他们总紧盯着别人的弱点和自己达不到的事上，这种消极被动的能量让他们的影响圈逐渐缩小。

所以拥有积极主动习惯的人，能不断扩他们的影响圈，从而走向成功。

## 习惯二：以终为始

> 如果你即将死去，你会希望你的家人、朋友、工作伙伴，甚至并不熟悉的人如何评价你？

这些评价就是你内心深处最重视的期待和价值观，把它们写下来成为你的生命宣言，作为衡量一切的标准。然后用自我意识去观察我们的一言一行，无论何时何地都遵循这些原则。

## 习惯三：要事第一

把所有你想做的、正在做的、没有做的事，按照重要和紧急两个维度写下来，然后按以下方式进行自我管理：

1. 识别超出我们影响圈的事，尽可能移除它们，它们只会浪费我们的时间和经历。
2. 培养预防型思维，防范于未然。
3. 不重要的事也鼓起勇气移除。
4. 剩下的大部分事项是最允许发挥创造力，也是产出和产能最为平衡的。把尽力都花在这些事情上，就能更加轻松地走向成功。

## 习惯四：双赢思维

人际关系的本质，是我们给对方开设一个情感账户，里面储存着依赖和安全感。当余额消耗殆尽，关系就会随之破裂。

人际交往过程中有6种相处模式：

<HSentence left>
利人利己、两败俱伤、损人利己<br>
独善其身、舍己为人、好聚好散
</HSentence>

| 模式       | 描述          | 结果     |
| -------- | ----------- | ------ |
| 利人利己     | 共同获益        | 最佳关系储备 |
| 两败俱伤     | 双方损失        | 关系破裂   |
| 损人利己     | 自我得益，伤害他人   | 长期关系亡  |
| 独善其身     | 自我保护，不关他人事  | 孤立无援   |
| 舍己为人     | 单方面付出，不求回报  | 容易被利用  |
| 好聚好散     | 临时合作，随时各奔东西 | 短期效益   |


人际交往错综复杂，不同的情况往往需要做出不同的选择，但只有利人利己这一种模式，才能不损害双方的情感账户。所以任何的关系，充足的情感账户余额或许能够允许短期信任的消耗，但长远来看如果不能做到双赢，就一定会两败俱伤。

只有带着双赢的思维，把生活看作一个合作的舞台，而不是角斗场，才是相互依赖的环境中唯一可行的交往模式。

## 习惯五：知彼解己

我们总在学习表达，却从来没有人来教我们如何去听。在沟通中，大多数人都只会不断表达自己的感受，然而倾听却是件更重要的事。

<HSentence>因为只有感同身受，别人才会向你吐露心声。</HSentence>

如果你想对人有影响力，首先你要了解他，而做到这一点不能只靠技巧。如果对方察觉到你在使用某种技巧，就会感到受骗和被操纵，自然不会敞开心扉。所以有效的沟通习惯，首先要有真诚开怀的人格，在此基础上培养移情聆听的技巧，才能建立情感账户实现安全交流。

## 习惯六：统合综效

统合终效的基本心态是：如果一位聪明有才的人跟我意见不同，那么对方的主张必定有我尚未体会的奥妙，值得加以了解。

判断和尊重差异，是统合终效的精髓。重视不同个体的情绪和智力差异，关注重点就能由为自己辩护转向共同创造。产生认同感和向心力，从而实现1 + 1 = 3，甚至等于3000。

## 习惯七：不断更新

> 不断更新就是：保护并优化你所拥有的最重要的资产—— 你自己。

包含四个方面：
<HSentence>身体、精神、智力、情感</HSentence>

更新和完善自我的过程，就是平衡和兼顾这四个要素的过程。

---

只有在身心平衡的状态下，才能保持这七个习惯，相辅相成、相互推动，螺旋式地完成自我提升和完善。
