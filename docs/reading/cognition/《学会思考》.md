> 我们身边似乎总有一些更聪明，看问题总是能看到根源和本质的人，他们是怎么做到的？

<!-- more -->

这不仅仅是因为他们逻辑清晰，而是头脑中有一套想问题的思维模型，这个模型有些人天生就容易形成，有些人则需要通过后天训练。

_这个思维就叫做批判性思维，该怎么形成？_

今天分享的这本书叫做《学会思考》，作者是“C计划”的三位联合创始人，这个计划是专门针对批判性思维教育的一个机构。这本书提炼了许多年来沉淀的许多批判性思维课程的精华总结。关于该怎么塑造自己的批判性思维，书里把它总结成六个步骤：

<HSentence>定位问题 → 获取信息 → 提炼信息 → 评价信息 → 应用信息 → 表达信息</HSentence>

这六个步骤相互关联，构成一个完整的批判性思维流程。

## 批判性思维

> 首先要知道，什么是批判性思维？

回想一下刷短视频的时候，如果你看到一些让人非常气愤的八卦或者新闻，比如说虐待动物、抛妻弃子，又或者一些娱乐圈翻车的明星艺人，你一般会怎么做？

大多数人可能第一时间会先发表一下自己的气愤，如果还是气不过就跑到当事人的主页骂一顿。但你发现没有，我们吃过的大多数瓜，十有八九都会反转，一次又一次的网暴就是这样形成的。

又或者工作中开会的时候，往往都是你一嘴我一嘴，最后很难取得统一的结论。和别人吵架的时候，往往总会事后才觉得自己没发挥好等等，这一系列看起来很小的事情，背后总是缺乏批判性思维的体现。面对一系列信息，我们第一时间总会去屈从自己的情绪，快速得出结论，然后草率地表达。 

_所以什么是批判性思维？_

书里说：<HSentence>批判性思维有两个关键，一是“信什么”，二是“做什么”。</HSentence>

- “信什么”的意思是，接收到信息的时候，不管是听人说话、读书还是刷短视频，不会轻易跟随自己或者大众的情绪，而是能够更加独立判断：“我要不要相信它”。
- “做什么”的意思是，接收信息后，也能够更独立地决定自己该说什么、做什么。

这两件事听起来很容易，但做起来却相当难。因为情绪与思维惯性的阻碍，经常让我们在实际中难以保持理性与独立。

丹尼尔·卡尼曼在《思考，快与慢》里说：<HSentence>第一系统远远强大过第二系统。</HSentence>

也就是说情绪的能量，远远大过理。这意味着人们往往不自觉地会先用“直觉”和“情绪”做判断，而不愿花力气推理或验证。该怎么克服这个问题呢？

书里拆分成12个问题，总结起来形成这六个步骤：

```markmap
# 独立思考全流程
## 定位问题
- 四步法
## 获取信息
- 信息核真
## 提炼信息
- 论证与论证结构
## 评价信息
- 立：ARG三标准
- 破：常见逻辑谬误
## 应用信息
### 是什么：归纳论证
### 怎么样
- 价值判断三段论
- 权衡论证
### 为什么：溯因论证
- 结构化归因
- 实验法
### 怎么办：实践论证
- 目标 - 手段
- 赋值法
## 表达信息
- 金字塔模型
- MECE原则
```

## 定位问题

很多人在接收到问题之后，都会去立刻讨论解决办法。但想要解决问题，其实最重要的是定位问题。

::: info 比如说：有一天你们公司一个大客户突然跑了，你的老板拉着你和一帮人一起开会。
- A说：跑了就跑了，他们特别难伺候。
- B说：那么大的单子决定公司的死活啊。
- C说：老板你确定他们不和我们合作吗？我们昨天还在沟通呢。
- D说：要不我们找人再去沟通一下吧
- 这时候A又说：得了，你可不知道伺候他们有多麻烦。
- 这时候B又说：我觉得现在，我们得先反思一下问题出在哪里。
然后，大家就这样你一言我一语，各执一词地讨论。
:::

如果这个过程中有一个人思路非常清晰，能够带着所有人往一个方向思考，这个人基本上就能成为团队中领导力和影响力最大的人，该怎么引导？

书里分享了一个“四步法”，生活中大多数问题，都可以用这个四步法去解决。

分别是：<HSentence>是什么 → 怎么样 → 为什么 → 怎么办</HSentence>

::: info 比如上面例子中混乱的讨论，根据这四步就可以拆分成四个问题逐一交流，一旦聊偏离了就及时拉回来。
- 是什么：客户是不是真的已经和别的公司签约？
- 怎么样：如果是真的，会给公司带来多大影响？
- 为什么：客户为什么没有续约？
- 怎么办：该怎么挽回损失？
:::

对这四步的含义如下所示：

### 是什么
 - 针对要讨论的现象，提供最基本的事实信息。

### 怎么样
 - 评估这件事的影响、后果、意义，对这件事做出价值判断^[价值判断：基于一个人的价值观，对某个事物做出好坏、对错、应否的判断。]。

### 为什么
 - 如果对一件事做出了负面评价 —— 这个问题究竟是怎么造成的？
 - 如果对一件事做出了正面评价 —— 它成功或令人欢喜、满意的原因是什么？

### 怎么办
 - 针对“为什么”层面找到的原因，给出具体的解决措施。确定接下来应该做什么去解决问题，或者如何进一步推广成功的经验。


## 更多步骤

完成第一步定位问题之后，下一步开始获取信息，然后开始提炼有用的信息，最后加以应用。在这个过程中我们还会遇到很多问题，比如这个信息爆炸的时代，网络上掺杂的信息质量参差不齐，该怎么学会筛选？筛选出一系列有用的问题和信息后，该怎么去识别出背后的规律？沟通过程中，遇到三观不合的人该怎么协调？要怎么更清晰地表达自己的观点等等。

这一系列问题，书中都像上面讲定位问题一样给我们整理出一个个思维模型，这一个个模型哪怕只是学好其中一个，都能对我们思维的准确性有很大的提升。当我们逐渐掌握这些模型，就能从容地应对信息爆炸、复杂决策和人际争端，在思考和沟通中保持更高质量的理性判断。

```markmap
# 阿芒未获得晋升
- 不知：
  不知晋升的标准
  - 内因：
  不善于总结、沟通，不清楚领导预期
  - 外因：
  公司缺乏清晰的、透明的晋升通道和机制

- 不能：
不能够得到晋升
  - 内因：
  自身能力有限，达不到标准
    - 内因：
    缺乏学习能力或意愿
    - 外因：
    学校/公司缺乏相应的培训
  - 外因：
  外部障碍（如领导使绊子）

- 不愿： 
不愿意晋升
  - 在当事人看来为此现状利大于弊
    - 功利实用：
    更高的职级压力更大、应酬更多，无法平衡事业和生活
    - 审美偏好：
    晋升为管理人员，但自己对管理工作不感兴趣
```

```markmap
- 接收/获取一条信息
`谁说的？`
  - 路径一：
  追溯信息源
    - 有明确的信息源
    `这些信息是原话吗？`
      - 能追溯到原始信息
      `说话的人是可靠的信息来源吗？`
        - 可信度高
          - 可采信/进一步验证
        - 可信度低
          - 不轻信/不转发
      - 不能追溯到原始信息
        - 不轻信/不转发
    - 无明确的信息源
      - 不轻信/不转发
  - 路径二：
  交叉验证
    - `谁是知道可靠信息的人？`
    `他们是怎么说的？`
```
