## 1. Install Apple Development Certificates

> Xcode → Settings → Accounts → Manage Certificates → Export Certificate

## 2. View Organization Unit

> Keychain Access → Apple Development: XXXX(ZXXX) → Show Profile → Details → Issuer Name → Organization Unit (G3)

## 3. Install Apple Intermediate Certificates

> [ Apple PKI](https://www.apple.com/certificateauthority/) → Apple Intermediate Certificates → Worldwide Developer Relations - G3 → Download
